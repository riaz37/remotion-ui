RemotionUI is a **registry-first component kit** for Remotion. Install compositions with the CLI — every file copies into your repo as **source you own**.

## How it works

1. **Initialize** — `npx remotion-ui init` scaffolds aliases and `remotion-ui.json`
2. **Add** — `npx remotion-ui add social-clip` copies source into `src/remotion/`
3. **Render** — import locally and render with Remotion
