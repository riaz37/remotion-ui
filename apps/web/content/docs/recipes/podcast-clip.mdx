# Podcast Clip

Use this when the user wants a podcast or interview clip with audio visuals and captions.

## Install

```bash
npx remotion-ui@latest add --recipe podcast-clip
```

<RenderCommand compositionId="PodcastClip" />

## Agent instructions

- Use `PodcastClip` for a complete composition.
- Use `AudioPulse`, `WaveformLine`, or `AudiogramScene` for custom audio visuals.
- Require an audio source and caption data for a synced final result.
- Keep captions readable and inside safe areas.

## Typical imports

```tsx
import { PodcastClip } from "@/compositions/podcast-clip";
import { AudioPulse } from "@/remotion/primitives/audio-pulse";
import { WaveformLine } from "@/remotion/primitives/waveform-line";
```
