## Usage notes

- Use `PodcastClip` for a complete composition.
- Use `AudioPulse`, `WaveformLine`, or `AudiogramScene` for custom audio visuals.
- Provide an audio source and caption data for a synced final result.
- Keep captions readable and inside safe areas.
- Import installed components from local source paths, not from `remotion-ui`.

## Agent notes

- Prefer this recipe when the user wants an audio-first clip with waveform and captions.
