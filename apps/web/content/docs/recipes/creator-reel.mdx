# Creator Reel

Use this when the user wants a creator-style vertical clip: hook, talking head, comment callout, B-roll, and CTA.

## Install

```bash
npx remotion-ui@latest add --recipe creator-reel
```

<RenderCommand compositionId="CreatorReel" />

## Agent instructions

- Use `CreatorReel` for the full 9:16 template.
- Pass `mediaSrc`, `audioSrc`, and `captions` when available.
- Customize `hookHeadline`, `talkingHeadTitle`, and `comment` for the user's story.
- Import installed components from local source paths, not from `remotion-ui`.

## Typical imports

```tsx
import { CreatorReel } from "@/compositions/creator-reel";
```
